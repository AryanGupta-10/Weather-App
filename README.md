# Weather-App
🌦️ A responsive weather app built with HTML, CSS &amp; JavaScript that fetches real-time weather data using API.

A simple and responsive weather application built using **HTML, CSS, and JavaScript** that allows users to check real-time weather conditions for any city.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Displays temperature, humidity, and weather condition
* ☁️ Dynamic weather icons based on conditions
* ⚡ Real-time data using Weather API
* 📱 Fully responsive design (works on mobile & desktop)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Weather API (e.g., OpenWeatherMap)

## ⚙️ How It Works

1. User enters a city name
2. App sends request to weather API
3. API returns real-time weather data
4. Data is displayed on the UI dynamically

---

## 🧪 How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/weather-app.git
   ```

2. Open the project folder:

   ```bash
   cd weather-app
   ```

3. Open `index.html` in your browser

---

## 🔑 API Setup

If you're using an API like OpenWeatherMap:

1. Get your API key
2. Replace it in your JavaScript file:

   ```js
   const apiKey = "YOUR_API_KEY";
   ```
